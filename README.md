# Predicting Urban Bike Sharing Demand Using Weather, Seasonal, and Calendar-Based Machine Learning Features

## Project Overview
This MSc dissertation project focuses on predicting hourly bike-sharing demand using machine learning regression models. The project uses the Seoul Bike Sharing Demand dataset, which contains hourly bike rental counts along with weather, seasonal, and calendar-based features.

## Dataset
Dataset: UCI Seoul Bike Sharing Demand  
Link: https://archive.ics.uci.edu/dataset/560/seoul+bike+sharing+demand

## Models
The project will compare the following regression models:
- Linear Regression
- Random Forest
- Gradient Boosting
- XGBoost

## Evaluation Metrics
The models will be evaluated using:
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score
- Actual-versus-predicted plots

## Project Management
Progress will be tracked using:
- Gantt chart
- Meeting minutes
- Weekly progress log
- GitHub commit history

## Tools and Technologies
- Python
- Google Colab / Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- XGBoost

## Expected Outcome
The project aims to identify the best-performing machine learning model for predicting hourly bike-sharing demand and explain how weather, seasonal, and calendar-based features influence bike rental patterns.
